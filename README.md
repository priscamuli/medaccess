# MedAccess Dashboard

MedAccess is a full‑stack web application designed to improve healthcare access and visibility in Kenya. It provides role‑based dashboards for **patients** and **clinics**, enabling real‑time updates on medicine availability and empowering patients to make informed decisions about where to seek care.

---

## 🚀 Features

- **Role‑Based Authentication**: Patients and clinics have distinct dashboards.
- **Clinic Dashboard**: Admins update medicine stock levels; changes sync instantly.
- **Patient Dashboard**: Interactive map shows nearby clinics with color‑coded availability (green = stocked, red = shortage).
- **Real‑Time Sync**: Firestore ensures updates are reflected live without refresh.
- **Responsive UI**: Built with TailwindCSS for clean, judge‑ready design.

---

## 🛠️ Tech Stack

- **Frontend**: React + TypeScript
- **UI**: TailwindCSS + custom components
- **Backend/Database**: Firebase Firestore
- **Authentication**: Firebase Auth (email + password)
- **Deployment**: Vercel / Netlify
